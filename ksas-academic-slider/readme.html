KSAS Homepage Slider

Description

Creates a custom post type for a homepage slider. The content area is the test that will display next to the photo. The photo must be 670px by 360px.

There is a metabox for selecting the background color (only matters for Academic Template - Blue), setting the slide destination, and if you would like to display the "Find Out More" button.

Your slides will be saved for future use if you deactivate the plugin, and will reappear upon activating the plugin

Template Compatibility

  • Academic Template

Permissions

The slide post type is only available to Super Admins and Site Admins. Permission to can be given to LAN Admins or other user types on a case by case basis.

Installation

Just activate the plugin on the desired site.
