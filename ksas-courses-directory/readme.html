KSAS Courses Directory

Description

Creates a custom post type for courses. Includes fields for: title, course description, number of credits, instructor name, prerequisites, course website, course times, and class limit.

You must designate the course as either Undergraduate, Graduate, or both for the course to display properly.

Remember to designate your academic department for each course. This will allow for better searching and cross listing

To create a course display page:

  1. Create a new page
  2. The page url must either be 'graduate-courses' or 'undergraduate-courses'. You can edit this under the Title field.
  3. Choose "Courses" from the "Template" drop down on the right side.
  4. Any content you enter in the editor will display above the course listings.
  5. Publish your page

Template Compatibility

  • Academic Template

Permissions

Courses are available for editing to all user types, but Contributors cannot publish. This privilege can be given on a case by case basis by all higher user types including LAN admins.

Installation

Just activate the plugin on the desired site
