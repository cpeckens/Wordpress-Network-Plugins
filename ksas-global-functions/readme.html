KSAS Global Functions

Description

Houses all the custom functions that apply to all sites

Functions include:

  • Preventing login erros
  • Blocking malicious queries
  • Removing Wordpress version number and other unnecessary info from the <head>
  • Removing unnecessary auto-generated classes from menus and navigation
  • Removes admin bar from front end
  • Registers and populates the Academic Department and Affiliation taxonomies
  • Houses custom post type UI functions (generated from Easy Custom Post Types plugin export)

Your slides will be saved for future use if you deactivate the plugin, and will reappear upon activating the plugin

Permissions

This is a network activated plugin. Available only to Super Admins

Installation

Network activate this plugin
